# **SHOW SUBSCRIPTIONS**

## **语法说明**

返回所有发布名、发布实例名、发布的数据库名、发布给该租户的时间、订阅名和创建该订阅的时间。

## **语法结构**

```
SHOW SUBSCRIPTIONS [ALL];
```

## **语法解释**

- **ALL** 选项可以看到所有有权限的订阅，未订阅的 sub_time, sub_name 为 null，不加 **ALL** 只能看到已订阅的发布信息。

## **示例**
